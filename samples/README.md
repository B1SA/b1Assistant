# Usage samples

Here you will find examples of conversations supported by B1 assistant.
All scenarios in here should be previously started by either

[U] = User
[A] = Alexa

```
[U]: Alexa, open B1
[A]: Hi, I am B1 Assistant. How can I help?
[U]: <Some Action> 
```

or
```
[U]: Alexa, ask be one <Some action>
```
